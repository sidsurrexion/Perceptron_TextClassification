1. Open the terminal or command prompt (in Windows).

2. Let us say the file is in /home/username/workspace/Perceptron/src and you are in the 
 home folder as of now then enter cd workspace/Perceptron/src to reach the destination.

3. On arriving at this folder in terminal type javac Driver.java.

4. It creates the class files. 

5. Type the command java Driver to execute the program which will report the 
output.

6. On run window enter the number of iterations first and the learning rate last by space being 
in between and execute for multiple combinations.
